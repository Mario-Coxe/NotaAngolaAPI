# NotaAngola
NotaAngola  é um sistema Sistema Multitenancy (SaaS) para Gestão de Notas


Neste projeto, estamos propondo o desenvolvimento de um sistema multitenancy (SaaS) para
a gestão de notas e a comunicação entre instituições acadêmicas, professores, alunos e pais. Sabemos
que atualmente é essencial adotar sistemas de gestão escolar para melhorar os processos, facilitar a
comunicação e aumentar a eficiência da administração acadêmica.
Nosso sistema será voltado especificamente para o gerenciamento de notas de estudantes em
várias instituições acadêmicas. Ele permitirá que os professores cadastrem as notas, os alunos
acessem suas próprias notas e os pais acompanhem o desempenho de seus filhos. Além disso, o
sistema contará com recursos inteligentes para auxiliar no cálculo das notas, gerar boletins, criar
quadros de honra e classificar os alunos.
A criação desse sistema trará muitas vantagens. Primeiro, as informações das notas serão
centralizadas em um único local, facilitando o acesso e a consulta por parte de alunos, pais e
professores. Além disso, a plataforma oferecerá recursos inteligentes que agilizarão o cálculo das
notas e a geração de boletins, otimizando o tempo dos professores e melhorando a eficiência dos
processos acadêmicos.
Nossos objetivos com o desenvolvimento desse sistema são os seguintes:
1. Desenvolver um sistema multitenancy (SaaS) que permita a gestão de notas de estudantes em
diferentes instituições acadêmicas.
2. Criar uma interface intuitiva e fácil de usar para que os professores possam cadastrar notas de
forma rápida e precisa.
3. Permitir que os alunos acessem suas próprias notas de maneira simples e atraente visualmente.
4. Facilitar a comunicação entre pais e professores por meio de um chat dedicado a cada turma,
possibilitando a interação e o acompanhamento das notas dos alunos.
5. Implementar recursos inteligentes que auxiliem os professores no cálculo das notas e na
geração de boletins, promovendo maior agilidade e precisão no processo.
6. Gerar quadros de honra da escola e da sala, destacando os alunos com melhores desempenhos
acadêmicos.
7. Realizar uma classificação dos alunos mais destacados em âmbito escolar, com base nas notas
obtidas.
8. Integrar um calendário no sistema, permitindo que os alunos visualizem prazos importantes,
como datas de provas, entrega de trabalhos e eventos escolares. Os professores tambémpoderão usar o calendário para marcar datas de aulas especiais, reuniões ou atividades
extracurriculares. Isso ajudará na organização dos alunos e no acompanhamento das
atividades acadêmicas.
9. O sistema irá incluir recursos automatizados de cálculo de notas, permitindo que os
professores insiram os critérios de avaliação e pesos para cada componente, e o sistema
calculará as notas finais automaticamente.
10.
Além de fornecer acesso às notas atuais, o sistema poderá armazenar o histórico de
notas de cada aluno. Isso permitirá que os alunos, pais e professores acompanhem o progresso
ao longo do tempo e identifiquem tendências ou áreas que necessitam de melhoria.
11.
O sistema oferecerá a opção de gerar relatórios personalizados, como boletins
individuais ou relatórios de turma, contendo informações sobre as notas dos alunos. Esses
relatórios podem ser gerados no formato digital ou impresso(Baixados).
12.
O sistema oferecerá análises e gráficos visuais para ajudar os professores, alunos e
pais a entenderem o desempenho do aluno em diferentes áreas ou disciplinas. Isso pode ajudar
a identificar pontos fortes e áreas que precisam de mais atenção.
13.
O sistema enviará alertas e notificações automáticas para os alunos e pais sobre novas
notas lançadas, médias alcançadas ou quaisquer atualizações relevantes relacionadas às notas.
Para o desenvolvimento do sistema, utilizaremos tecnologias modernas de desenvolvimento
web e banco de dados. A arquitetura será projetada de forma escalável e segura, garantindo a
privacidade e a integridade dos dados dos alunos. A interface do sistema será construída com foco na
usabilidade, utilizando princípios de design responsivo para oferecer uma experiência agradável tanto
em dispositivos móveis quanto em computadores desktop, será uma PWA.
Planejamos testar o sistema em diferentes instituições acadêmicas para garantir sua
adaptabilidade e funcionalidade em diferentes contextos educacionais. Além disso, coletaremosfeedback dos usuários, como professores, alunos e pais, para aprimorar continuamente o sistema e
atender melhor às necessidades de todos os envolvidos.
Com todos esses recursos adicionais, nosso sistema multitenancy se tornará uma plataforma
abrangente e completa para a gestão de notas, a comunicação entre os envolvidos. Acreditamos que
essa solução contribuirá para a melhoria do desempenho acadêmico dos alunos, além de facilitar a
interação entre escola, professores e pais, fortalecendo o acompanhamento do desenvolvimento dos
estudantes ao longo de sua jornada educacional.
Em resumo, estamos entusiasmados com o desenvolvimento desse sistema multitenancy para
gestão de notas para diferentes instituições acadêmicas. Acreditamos que essa solução trará inúmeras
vantagens para a área educacional, centralizando as notas, oferecendo recursos inteligentes e
facilitando a comunicação com os pais.
